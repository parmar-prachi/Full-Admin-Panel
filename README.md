# 🚀 UserHub - Admin Panel

A modern **User Management Admin Panel** built using **Node.js, Express.js, MongoDB, EJS, and MVC Architecture**.

This project allows an administrator to manage users efficiently with authentication, image uploads, and a responsive dashboard.
